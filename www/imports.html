Importation partielle

Si nous ne voulons pas convertire une feuille de style entière en SASS, nous pouvons créer des fichiers SASS partiels qui contiennent de petits extraits de CSS que vous pouvez inclure dans d'autres fichiers SASS. C'est une excellente façon de modulariser notre CSS et de nous aider à garder les choses plus faciles à entretenir par la suite.
Un partiel est simplement un fichier SASS nommé avec un Underscore avant le nom du fichier. Nous pouvons nommer _partial.scss. par exemple Le trait de soulignement permet à Sass de savoir que le fichier n'est qu'un fichier partiel et qu'il ne doit pas être généré dans un fichier CSS. Les partiels Sass sont utilisés avec la directive @import. 


//_colors.scss est importée
 @import "colors";
// Cela va importer foo.scss dans le
// dossier utilitaires (chemin relatif)
 @import "utils/foo.scss";

Importations imbriquées

Si demo.scss contient:


.example {
  color: red;
}

puis


#main {
  @import "demo";
}

Ce qui sera compilé en:


#main .example {
  color: red;
}
